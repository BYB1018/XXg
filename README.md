# 幸存者冒险 (Survivor Adventure)

一个受吸血鬼幸存者启发的2D生存Roguelike游戏。玩家需要在不断涌来的敌人中生存，收集经验升级，获得新的技能来变得更强。

## 游戏特性
- 单一场景的无尽生存模式
- 自动攻击系统
- 经验值与升级系统
- 技能选择系统
- 随机敌人生成
- 简单的碰撞检测

## 技术架构
- 使用Unity引擎开发
- 2D像素风格画面
- 使用C#编写游戏逻辑
